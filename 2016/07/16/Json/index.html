首先，json 数据的书写格式是：名称/值对，和java中map很类似，但它的key只能用string，如”firstName” : “John”，json值可以为：数字（整数或浮点数），字符串（在双引号中），逻辑值（true 或 false），数组（在方括号中），对象（在花括号中），null。
JSON 对象在花括号中书写，对象可以包含多个名称/值对，如：{ “firstName”:”John” , “lastName”:”Doe” }
